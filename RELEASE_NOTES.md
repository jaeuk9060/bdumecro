# BDU LMS 트래커 v1.0.0

부경대학교 LMS 강의 자동 시청 도구입니다.

---

## 주요 기능

### 자동 강의 시청
- **전체 자동 시청**: 미청취 강의가 있는 모든 과목을 순차적으로 자동 시청
- **과목별 시청**: 원하는 과목만 선택하여 해당 과목의 미청취 강의 시청
- **출석 확인 자동 처리**: 강의 중 출석 확인 모달 자동 클릭
- **이어듣기 자동 처리**: 이어듣기 모달 자동 "예" 클릭

### 직관적인 GUI
- 과목별 진도율 시각화 (프로그레스 바)
- 미청취 강의 수 표시
- 실시간 상태 메시지
- 미청취 강의가 없는 과목 자동 숨김

### 안전한 로그인
- 자동 로그인 없음 (직접 로그인 필요)
- 개인정보 저장 없음

---

## 설치 방법

### 요구사항
- Windows 10/11
- Chrome 브라우저 (설치 필수)
- 인터넷 연결

### 실행 방법
1. `BDU_LMS_Tracker_v1.0.0.zip` 다운로드
2. 원하는 위치에 압축 해제
3. `BDU_LMS_Tracker.exe` 실행

> ChromeDriver는 자동으로 다운로드됩니다 (webdriver-manager 사용)

---

## 사용 방법

### 1단계: 포털 열기
1. **[포털열기]** 버튼 클릭
2. Chrome 브라우저가 열리고 부경대 포털로 이동
3. **직접 로그인** (ID/PW 입력)

### 2단계: 강의 목록 확인
1. 로그인 후 **[강의페이지]** 버튼 클릭
2. LMS 강의 목록 페이지로 이동
3. 미청취 강의가 있는 과목들이 화면에 표시됨

### 3단계: 강의 시청

**방법 A: 전체 시청 (권장)**
- **[전체시청]** 버튼 클릭
- 모든 미청취 강의를 순차적으로 자동 시청

**방법 B: 과목별 시청**
- 원하는 **과목 카드** 클릭
- 해당 과목의 미청취 강의만 시청

---

## 주의사항

1. **시청 중 브라우저 조작 금지**
   - 자동 시청 중에는 브라우저를 건드리지 마세요
   - 강의 전환이 자동으로 이루어집니다

2. **Chrome 브라우저 필수**
   - Chrome이 설치되어 있어야 합니다
   - Edge, Firefox는 지원하지 않습니다

3. **인터넷 연결 유지**
   - 안정적인 인터넷 연결이 필요합니다

---

## 문제 해결

| 문제 | 해결 방법 |
|------|-----------|
| 브라우저가 열리지 않음 | Chrome 설치 확인, 재시작 |
| 과목이 표시되지 않음 | 로그인 확인, 새로고침 버튼 클릭 |
| 강의가 재생되지 않음 | 팝업 차단 해제, 브라우저 재시작 |
| "ChromeDriver 오류" | 인터넷 연결 확인 (자동 다운로드 필요) |
