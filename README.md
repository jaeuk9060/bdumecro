# BDU LMS 트래커

부경대학교 LMS 강의 자동 시청 도구입니다.

---

## 설치 방법

### 1. Python 설치
- Python 3.11 이상 필요
- https://www.python.org/downloads/ 에서 다운로드

### 2. 의존성 설치
```bash
pip install -r requirements.txt
```

### 3. 실행
```bash
python src/main.py
```

### 버튼 설명

| 버튼 | 색상 | 기능 |
|------|------|------|
| **포털열기** | 파랑 | 브라우저를 열고 LMS 로그인 페이지로 이동 |
| **강의페이지** | 주황 | LMS 강의 목록 페이지로 이동 |
| **전체시청** | 보라 | 미청취 강의가 있는 모든 과목 자동 시청 |
| **새로고침** | 초록 | 과목 목록 새로고침 |

---

### 사용 순서

#### Step 1: 포털 열기
1. **[포털열기]** 버튼 클릭
2. 브라우저가 열리고 부경대 포털 로그인 페이지로 이동
3. **직접 로그인** (ID/PW 입력)
4. 로그인 완료 후 과목 목록이 자동으로 표시됨

#### Step 2: 강의 시청 (2가지 방법)

**방법 A: 전체 시청**
- **[전체시청]** 버튼 클릭
- 미청취 강의가 있는 모든 과목을 순차적으로 자동 시청
- 미청취가 0개인 과목은 자동으로 건너뜀

**방법 B: 과목별 시청**
- 원하는 **과목 카드 클릭**
- 해당 과목의 미청취 강의만 자동 시청

#### Step 3: 새로고침
- 시청 완료 후 **[새로고침]** 버튼으로 진도율 확인
- 완료된 과목(미청취 0개)은 목록에서 자동으로 숨겨짐

## 주의사항

1. **로그인은 직접 해야 합니다**
   - 보안상 자동 로그인은 지원하지 않습니다
   - 포털 열기 후 직접 로그인해주세요

2. **시청 중 브라우저 조작 금지**
   - 자동 시청 중에는 브라우저를 건드리지 마세요
   - 강의 전환이 자동으로 이루어집니다

3. **인터넷 연결 필요**
   - 안정적인 인터넷 연결이 필요합니다

4. **Chrome 브라우저 필요**
   - Chrome이 설치되어 있어야 합니다
   - ChromeDriver는 자동으로 관리됩니다

## 문제 해결

### 브라우저가 열리지 않아요
- Chrome이 설치되어 있는지 확인하세요
- 인터넷 연결을 확인하세요

### 과목이 표시되지 않아요
- 로그인이 완료되었는지 확인하세요
- [새로고침] 버튼을 클릭해보세요

### 강의가 재생되지 않아요
- 팝업 차단이 해제되어 있는지 확인하세요
- 브라우저를 닫고 다시 시도해보세요

### 오류가 계속 발생해요
- 프로그램을 완전히 종료 후 다시 실행하세요
- 브라우저 창을 모두 닫고 다시 시도하세요
